# 옴팡이 게임 (Ompangi Game)

캐릭터 "옴팡이"를 이용한 간단한 게임입니다. <br/>
손가락으로 화면을 탭하여 옴팡이를 이동시킬 수 있습니다. <br/>
화면을 탭하지 않으면 옴팡이는 오른쪽으로 이동하며, 화면을 탭하면 왼쪽으로 이동하게 됩니다.
