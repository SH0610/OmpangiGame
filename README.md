# 옴팡이 게임 (Ompangi Game)

캐릭터 "옴팡이"를 이용한 간단한 게임 <br/>

## How To Play
- 손가락으로 화면을 탭하여 옴팡이 이동 가능
- 화면을 탭하지 않을 경우, 옴팡이는 오른쪽으로 이동
- 화면을 탭할 경우, 옴팡이는 왼쪽으로 이동
- 60초의 제한 시간이 있으며, 제한 시간 내에 하트 5개를 모두 잃게 되면 게임 종료
